1. ГЕНЕРАЦИЯ ПРОЕКТА
Создает нужные файлы для сборки в отдельной папке build. 
Выполнять из корневой директории проекта:
> cmake -B build

(Опционально для конкретной версии студии: cmake -B build -G "Visual Studio 17 2022" -A x64)
Проект можно открыть в Visual Studio через файл CMakeLists.txt или сгенерированный .sln.

2. СБОРКА ПРОЕКТА
Компиляция исходного кода. Выполнять из корня проекта:
> cmake --build build --config Debug
(Для релизной версии замените Debug на Release)


ЗАПУСК И АРГУМЕНТЫ
--------------------------------------------------
Формат вызова:
program <input_file> <output_file> <type_proc> <param1> <param2>

<input_file>  - путь к входному файлу
<output_file> - путь к выходному файлу
<type_proc>   - режим работы программы (test или sort)

1. Режим генерации (test)
   Создает файл со случайными строками.
   <param1> - максимальная длина строки (число > 0)
   <param2> - количество строк (число > 0)
   
   Пример: program in.txt out.txt test 64 10

2. Режим сортировки (sort)
   Сортирует строки из входного файла.
   <param1> - тип сортировки:
              slow - сортировка пузырьком
              fast - быстрая сортировка
   <param2> - тип сравнения (от 1 до 4):
              1 - лексикографическая с учетом регистра
              2 - лексикографическая без учета регистра
              3 - по длине строки (затем лексикографическая)
              4 - лексикографическая с конца строки
              
   Пример: program in.txt out.txt sort fast 3

СПРАВКА
